nginx http 502 503 504区别

2017年1月22日

做web开发，常常回遇到5XX的状态码，区别记录下来：

  • 504：Gateway Timeout 网关超时，如php-fpm运行超时
  • 502：Bad Gateway 网关错误，如无空闲的php-fpm进程
  • 503：Service Unavailable 服务不可用，如当前并发请求数达到nginx limit_conn上线

504表示网关可用，但是处理超时
502表示没有可用的网关
503表示nginx不可提供服务
